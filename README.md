## Getting the R package
- Make sure to start a fresh R session an type:
  - `library(devtools)`
  - `install_github("blogsvoices/iSAX")`
- At this point you should have `iSAX` installed and can proceed:
  - `library(iSAX)`
- You should be ready to go!
